o que é html
- HTML (hyperText markup Lenguage -Linguagem de marcação de HyperTexto)
-não vira codigo binario
- Os comando HTML são chamados de TAGS. As tags sao palavras chave
que ficam cercadas pelos simbolos maior e menor < >. Ex:. <hr>.. <body>..<.body>
- As tags são dividias em duas tags simples e composta a tag simples não
precisa ser encerrada. Ex.: <hr> ... <br> ... <img> ... <meta> ...
jJá as tags compostas precisam ser encerradas. Ex: <body>... </body>
<title>...</title>

As tags tanto simples como composta possuem, atributos(propriedades).
EX.: <img src="foto.png" alt="foto do cliente">

<body background="red">... </body>
.
 Para transformar código em objetos, imagens, cores o navegador usa
 ferramentas chamada, em motor web. Ele realiza o processo
 de renderização, que é transformar codigo em objetos

 Todo o arquivo HTMl deve ser salvo com a extensão .html
 Para testar um arquivo html nós o abrimos em um navegador web
 (chrome, opera, edge, firefox, safari, yandex, brave, tor...)

 CSS, o que é?

 CSS-Cascad style shett (Folha de estilio em Cascata)
 O css é utilizado para fazer formatação em páginas HTMl.

 E você pode aplicar o css de 3 formas:
 - Arquivo externo css 
 - Embutir o css na tag head da pagina html
 - Aplicar diretamente na tag html desejada 

 Há também 3 elementos de aplicação:

 - ID indetificador, usamos hashtag #
 - Class clasese, usamos ponto.
 - Tag tag html
